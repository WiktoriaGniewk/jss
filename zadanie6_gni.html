Rodzaje okienek

  • Okienko alert – wyświetla okienko dialogowe z informacją i przyciskiem OK.
  • Okienko prompt – generuje okno, w którym użytkownik może wpisać wartość, zwracaną następnie do programu.
  • Okienko confirm – służy do tego, aby użytkownik coś potwierdził (stąd nazwa "confirm").

Konwersja liczb

  • Wczytanie zmiennej z użyciem prompt wykonywane jest jako ciąg znaków, dlatego konieczna jest konwersja z ciągu znaków na wartości liczbowe, aby można było wykonywać obliczenia.

Przy nadawaniu nazw zmiennym należy przestrzegać kilku zasad:

  • Nazwa zmiennej musi zaczynać się od litery, znaku "$" lub "_", a może zawierać tylko litery (bez polskich znaków), cyfry oraz "$" i "_".
  • Przykłady poprawnych nazw zmiennych: zmienna1, $wynik, _rozmiar, wynik1, Rekord, $_gracz
  • Przykłady błędnych nazw zmiennych: 1zmienna, wynik gry, wysokosc&szerokosc, wynik.rekord, &gracz, wysokość, imie+nazwisko.
  • Wielkość liter ma znaczenie: Rekord i rekord to dwie różne zmienne! Jest to bardzo częsta przyczyna błędów w kodzie początkujących programistów.
  • Nie należy używać słów kluczowych JavaScript jako nazw zmiennych.

Omówienie typów zmiennych w JavaScript

  • Number – Typ liczbowy: zmienne całkowite oraz zmiennoprzecinkowe.
  • String – Typ tekstowy.
  • Boolean – Typ logiczny (prawda/fałsz).
  • Null – Pusta zawartość (brak wartości).
  • Undefined – Typ niezidentyfikowany (zmienna bez przypisanej wartości).
  • Object – Obiekt, który może zawierać kolekcję właściwości i metod.
